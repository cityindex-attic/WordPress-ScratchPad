=== Tools Demo ===
Tags: widget, plugin, sidebar, title, url
version: 1.0.0
Store and display title,url in sidebar widget or templates.

== Description ==

Tools Demo is a handy WordPress plugin that allows you to save, edit and delete title,url into the sidebar (via widget) or page body (via template tags). The sidebar widget allows you to set an optional Title, and text header and footer. 

== Installation ==

IMPORTANT: This plugin uses the WP_Widget class introduced in WordPress v2.8 and will not work with earlier versions of WordPress.

1. Upload `tooldemo.php` and `tooldemo_admin.php` to the `/wp-content/plugins/` directory of your WordPress installation
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The Tools Demo widget can now be configured and used from the Appearance -> Widgets menu
4. Text entries can be managed via from the Settings -> Tools Demo

Note: During installation, Tools Demo creates a new database table to store   - you should see three tools records after installation by clicking on the Settings -> Tools Demo.
